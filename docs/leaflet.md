# Leaflet 

This project uses [Leaflet.js](https://leafletjs.com/) to create annotated maps and images.

For images it uses [CRS](https://en.wikipedia.org/wiki/Spatial_reference_system) to add points to the image.

- https://leafletjs.com/examples/crs-simple/crs-simple.html
- https://leafletjs.com/reference.html#crs
- https://leafletjs.com/reference.html#imageoverlay
- basemaps, https://leaflet-extras.github.io/leaflet-providers/preview/
